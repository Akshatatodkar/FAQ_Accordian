What is HTML?

HTML stands for HyperText Markup Language. It's a standard markup language used to create webpages. It structures the content on a webpage using elements and tags.

What is CSS?

CSS stands for Cascading Style Sheets. It's a stylesheet language used for describing the presentation and formatting of HTML documents.

How can I change the font size of text using CSS?

You can change the font size of text with the font-size property. For example: font-size: 16px; sets the font size to 16 pixels.

What is JavaScript?

JavaScript is a programming language used to make webpages interactive. It can be used to add functionality, manipulate the DOM (Document Object Model), and handle user interactions.
